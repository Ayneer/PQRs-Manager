Sistema de gestión PQRs


Hola, la compañia ha dejado este sistema para su uso, el cual le permitirá realizar PQRs de tipo Petición, Queja o Reclamo.

También podrá hacer consultas sobre las PQRs realizadas, solo debe tener el número de radicado, entregado al final de cada creación.

Nota: Solo se puede registrar un reclamo sobre una petición o queja realizada hace más de 5 días, o que tenga respuesta del área administrativa.
